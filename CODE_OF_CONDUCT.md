# Code of Conduct

## Our Pledge
We are committed to providing a friendly, safe, and welcoming environment for everyoneâ€”regardless of background, identity, or experience level.

## Expected Behavior
- Be respectful and constructive
- Assume good intent
- Keep discussions focused on the project
- Give actionable feedback when possible

## Unacceptable Behavior
- Harassment, hate speech, or discrimination
- Personal attacks or deliberate intimidation
- Spam, trolling, or disruptive behavior
- Sharing private information (doxxing)

## Scope
This Code of Conduct applies to:
- GitHub Issues
- GitHub Discussions
- Pull Requests
- Any community spaces associated with ForgeDesk

## Enforcement
Project maintainers may:
- Edit or remove content
- Lock conversations
- Temporarily or permanently restrict participation

## Reporting
If you experience or witness unacceptable behavior, please open a private report by contacting the maintainers through GitHub.

Thank you for helping keep ForgeDesk professional and welcoming.
